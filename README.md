# 🛍️ NewStore

Projeto de e-commerce desenvolvido com foco em prática de front-end utilizando HTML, CSS e JavaScript.

O objetivo do projeto é simular uma loja virtual moderna, trabalhando conceitos importantes como manipulação do DOM, organização de componentes, consumo de API, carrinho de compras e armazenamento de dados no navegador.

---


# 🚀 Funcionalidades

* ✅ Listagem dinâmica de produtos
* ✅ Consumo de API de produtos
* ✅ Criação dinâmica de cards
* ✅ Carrinho de compras
* ✅ Adicionar produtos ao carrinho
* ✅ Remover produtos do carrinho
* ✅ Navegação entre páginas
* ✅ Armazenamento utilizando LocalStorage
* ✅ Layout responsivo
* ✅ Interface moderna

# 🛒 Como Executar o Projeto

## 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/newstore.git
```

## 2. Abra a pasta do projeto

```bash
cd newstore
```

## 3. Execute o projeto

Basta abrir o arquivo `index.html` no navegador.
